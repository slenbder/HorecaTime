# HorecaTime Bot

Telegram-бот для учёта рабочего времени сотрудников HoReCa (рестораны, бары, кухня).

## Статус проекта

- ✅ Модуль авторизации (регистрация, одобрение заявок, кеш ролей)
- 🔨 Главное меню
- 🔨 Модуль внесения смен
- 🔨 Админские команды

## Технологии

- Python 3.11+
- aiogram 3.x
- Google Sheets API
- SQLite

## Установка

```bash
python -m venv .venv
source .venv/bin/activate  # macOS/Linux
pip install -r requirements.txt
