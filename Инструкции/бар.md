# 📋 Инструкция — Отдел Бар

> Это руководство для сотрудников отдела **Бар**: Барменов и Барбэков.  
> Если ты только что получил доступ к боту — ты по адресу.

---

## Что такое этот бот и зачем он нужен

**HorecaTime** — Telegram-бот для учёта рабочего времени. Через него ты:
- вносишь свои рабочие смены
- отдельно фиксируешь тусовочные часы
- видишь сколько часов отработано и сколько заработано

Всё общение — прямо в Telegram. Никаких дополнительных приложений устанавливать не нужно.

> **Важно:** если бот завис или не отвечает на команду — напиши `/cancel`. Это сбросит любое незавершённое действие. Иначе - обратись к разработчику.

## Бармен

### Доступные команды

| Команда | Что делает |
|---|---|
| `/shift` | Внести рабочую смену |
| `/hours_first` | Часы и заработок за 1–15 число текущего месяца |
| `/hours_second` | Часы и заработок за 16–конец месяца + итог |
| `/hours_last` | Часы и заработок за прошлый месяц |
| `/schedule` | Получить график работы (PDF-файл) |
| `/sheet` | Ссылка на таблицу с графиком |
| `/contact_dev` | Написать разработчику |

---

### Как внести смену

У Бармена два шага: сначала основная смена, затем тусовочные часы.

---

#### Шаг 1 — вводишь дату и время смены

Напиши `/shift`. Бот ответит:

```
Введите смену:

13.03 10:00-20:00

Можно отменить ввод командой /cancel
```

Отправь смену в формате **ДД.ММ ЧЧ:ММ-ЧЧ:ММ**.

Примеры:
- `13.03 16:00-00:00`
- `01.04 18:00-02:00` *(смена через полночь — поддерживается)*

Бот посчитает количество часов и сразу предложит ввести тусовочные:

```
⏱ Смена 13.03.26: Часы смены = 8 ч

Были тусовочные часы?
Введите диапазон или 0:

00:00-04:00
```

Бот подсказывает пример диапазона: начало = конец твоей смены, конец = +4 часа от него.

---

#### Шаг 2 — тусовочные часы

**Если тусовочных не было** — введи `0`.  
Смена записывается сразу:

```
✅ Смена 13.03.26 записана
Часы смены = 8 ч
```

**Если тусовочные были** — введи диапазон времени в формате **ЧЧ:ММ-ЧЧ:ММ**.

Пример: `00:00-03:00`

Бот сам посчитает количество тусовочных часов (с точностью до 0.5 ч).


**Важно:** тусовочные часы **не должны пересекаться** с основной сменой. Если пересечение есть — бот сообщит об ошибке и попросит ввести диапазон заново:

```
❌ Тусовочные часы пересекаются с основной сменой (16:00–00:00).
Исправьте диапазон:
```

После успешного ввода смена записывается:

```
✅ Смена 13.03.26 записана
Часы смены = 8 ч | Тусовочные = 3 ч
```

### Как посмотреть часы и заработок

| Команда | Период |
|---|---|
| `/hours_first` | 1–15 число текущего месяца |
| `/hours_second` | 16–конец месяца + итог за весь месяц |
| `/hours_last` | Прошлый месяц |

У Бармена **две ставки**: базовая (основные рабочие смены) и повышенная (тусовочные часы). В отчёте видна разбивка:

```
📊 Первая половина месяца (1–15)
Отработано: 75 ч
Доп. часы: 15 ч
• 75 ч × 350 р = 26250 р
• 15 ч × 500 р = 7500 р (тусовочные)
💰 Итого: 33750 р
```

---

### Частые вопросы

**Бот не принимает диапазон тусовочных**  
Проверь, что диапазон не пересекается с основной сменой. Если смена закончилась в 00:00, тусовочные начинаются с 00:00.

**Смена заканчивается после полуночи — как вводить?**  
Так и вводи: `22:00-02:00`. Бот поддерживает переход через полночь.

**Внёс неправильные тусовочные**  
Самостоятельно изменить нельзя. Обратись к администратору.

**Перезаписал смену за дату, где уже было значение**  
Бот перезапишет ячейку. При перезаписи администратор получит уведомление со старым и новым значением.

## Барбэк

Барбэк работает **точно так же, как Бармен** — те же команды, тот же флоу внесения смены с тусовочными часами, те же принципы расчёта заработка.

Используй инструкцию выше в разделе [Бармен](#бармен) — всё актуально.

## 🛟 Общие советы для отдела Бар

- **Бот завис** → `/cancel` → начни заново
- **Ошибочная смена записана** → обратись к администратору
- **Не видишь кнопок** → обнови Telegram или напиши `/cancel`
- **Вопросы по ставке или заработку** → к администратору
- **Технические проблемы с ботом** → `/contact_dev`
