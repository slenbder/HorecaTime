# 📋 Инструкция — Кухня

> Это руководство для сотрудников **Кухни**: Руководящего состава, сотрудников цехов и дополнительных позиций (Грузчик, Закупщик).  
> Если ты только что получил доступ к боту — ты по адресу.

---

## Что такое этот бот и зачем он нужен

**HorecaTime** — Telegram-бот для учёта рабочего времени. Через него ты:
- вносишь свои рабочие смены
- видишь сколько часов отработано и сколько заработано

Всё общение — прямо в Telegram. Никаких дополнительных приложений устанавливать не нужно.

> **Важно:** если бот завис или не отвечает на команду — напиши `/cancel`. Это сбросит любое незавершённое действие.

## Руководящий состав

> Шеф-повара, су-шефы, бренд-шефы и другие руководящие позиции кухни.

### Доступные команды

| Команда | Что делает |
|---|---|
| `/shift` | Внести рабочую смену |
| `/hours_first` | Часы и заработок за 1–15 число текущего месяца |
| `/hours_second` | Часы и заработок за 16–конец месяца + итог |
| `/hours_last` | Часы и заработок за прошлый месяц |
| `/schedule` | Получить график работы (PDF-файл) |
| `/sheet` | Ссылка на таблицу с графиком |
| `/contact_dev` | Написать разработчику |

---

### Как внести смену

Напиши `/shift`. Бот ответит:

```
Введите смену или несколько смен:

13.03 10:00-20:00
```

Отправь одну смену или несколько — каждую на **новой строке**:

```
13.03 10:00-22:00
14.03 10:00-22:00
```

Бот запишет все смены сразу и подтвердит:

```
✅ Записано смен: 2
13.03.26: 12 ч
14.03.26: 12 ч
```

При одной смене:

```
✅ Смена 13.03.26 записана
Часы смены = 12 ч
```

> **Важно:** если хотя бы одна строка написана с ошибкой — не запишется **ни одна** смена. Бот сообщит об ошибке — исправь и отправь снова.

Доп. часов и фото на Кухне нет.

---

### Как посмотреть часы и заработок

| Команда | Период |
|---|---|
| `/hours_first` | 1–15 число текущего месяца |
| `/hours_second` | 16–конец месяца + итог за весь месяц |
| `/hours_last` | Прошлый месяц |

Пример ответа:

```
📊 Первая половина месяца (1–15)
Отработано: 120 ч
💰 Заработок: 54000 р
```

---

### Примечание про должность

При регистрации в боте ты вводил свою должность вручную (например: «Шеф ЗЦ», «Бренд-шеф», «Су-шеф КЦ»). Именно это название отображается в графике работы. Если в таблице написано что-то не то — обратись к администратору.

### Доступные команды

| Команда | Что делает |
|---|---|
| `/shift` | Внести рабочую смену |
| `/hours_first` | Часы и заработок за 1–15 число текущего месяца |
| `/hours_second` | Часы и заработок за 16–конец месяца + итог |
| `/hours_last` | Часы и заработок за прошлый месяц |
| `/schedule` | Получить график работы (PDF-файл) |
| `/sheet` | Ссылка на таблицу с графиком |
| `/contact_dev` | Написать разработчику |

---

### Как внести смену

Напиши `/shift`. Отправь одну смену или несколько (каждую на новой строке) в формате **ДД.ММ ЧЧ:ММ-ЧЧ:ММ**:

```
13.03 10:00-22:00
```

или несколько сразу:

```
13.03 10:00-22:00
14.03 10:00-22:00
15.03 11:00-21:00
```

Бот запишет и подтвердит. Если хотя бы одна строка написана с ошибкой — ни одна смена не запишется. Бот сообщит об ошибке — исправь и отправь заново.

---

### Как посмотреть часы и заработок

`/hours_first`, `/hours_second`, `/hours_last`.

Пример:

```
📊 Первая половина месяца (1–15)
Отработано: 108 ч
💰 Заработок: 32400 р
```

## 🛟 Общие советы для Кухни

- **Бот завис** → `/cancel` → начни заново
- **Ошибочная смена записана** → введи `/shift` заново за ту же дату с правильными данными — бот перезапишет запись
- **Перезаписал смену за дату, где уже было значение** → бот перезапишет ячейку; при перезаписи администратор получит уведомление со старым и новым значением
- **Не видишь кнопок** → обнови Telegram или напиши `/cancel`
- **Вопросы по ставке или заработку** → к администратору
- **Технические проблемы с ботом** → `/contact_dev`
