# 📋 Инструкция — МОП

> Это руководство для сотрудников подразделения **МОП**: Клининга и Котломоя.  
> Если ты только что получил доступ к боту — ты по адресу.

---

## Что такое этот бот и зачем он нужен

**HorecaTime** — Telegram-бот для учёта рабочего времени. Через него ты:
- вносишь свои рабочие смены
- видишь сколько часов отработано и сколько заработано

Всё общение — прямо в Telegram. Никаких дополнительных приложений устанавливать не нужно.

> **Важно:** если бот завис или не отвечает на команду — напиши `/cancel`. Это сбросит любое незавершённое действие.

---

## Клининг и Котломой

Флоу внесения смены одинаковый для обеих позиций.

### Доступные команды

| Команда | Что делает |
|---|---|
| `/shift` | Внести рабочую смену |
| `/hours_first` | Часы и заработок за 1–15 число текущего месяца |
| `/hours_second` | Часы и заработок за 16–конец месяца + итог |
| `/hours_last` | Часы и заработок за прошлый месяц |
| `/schedule` | Получить график работы (PDF-файл) |
| `/sheet` | Ссылка на таблицу с графиком |
| `/contact_dev` | Написать разработчику |

---

### Как внести смену

Напиши `/shift`. Бот ответит:

```
Введите смену или несколько смен:

13.03 10:00-20:00
```

Отправь одну смену или несколько — каждую на **новой строке** — в формате **ДД.ММ ЧЧ:ММ-ЧЧ:ММ**:

```
13.03 08:00-17:00
```

или сразу несколько:

```
13.03 08:00-17:00
14.03 08:00-17:00
15.03 08:00-16:00
```

> 📸 *Скриншот: пример ввода смены МОП*

Бот запишет и подтвердит:

```
✅ Смена 13.03.26 записана
Часы смены = 9 ч
```

При нескольких сменах:

```
✅ Записано смен: 3
13.03.26: 9 ч
14.03.26: 9 ч
15.03.26: 8 ч
```

> **Важно:** если хотя бы одна строка написана с ошибкой — не запишется **ни одна** смена. Бот сообщит об ошибке — исправь и отправь снова.

Доп. часов и фото нет.

---

### Как посмотреть часы и заработок

| Команда | Период |
|---|---|
| `/hours_first` | 1–15 число текущего месяца |
| `/hours_second` | 16–конец месяца + итог за весь месяц |
| `/hours_last` | Прошлый месяц |

Пример ответа:

```
📊 Первая половина месяца (1–15)
Отработано: 72 ч
💰 Заработок: 21600 р
```

### Частые вопросы

**Бот не реагирует на /shift**  
Напиши `/cancel` — это сбросит зависшее действие. После этого пробуй снова.

**Внёс смену с ошибкой**  
Самостоятельно изменить запись нельзя. Обратись к администратору Зала.

**Вопросы по ставке**  
Ставки устанавливает администратор. Обратись к администратору Зала.

## 🛟 Общие советы

- **Бот завис** → `/cancel` → начни заново
- **Ошибочная смена** → обратись к администратору Зала
- **Вопросы по ставке или заработку** → к администратору Зала
- **Технические проблемы с ботом** → `/contact_dev`
